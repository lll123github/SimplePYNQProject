gpio_in_out.zip 是运行在板子上的文件。开发板型号：背面标注小熊猫ZYNQ MINI，正面标注ZYNQ ECO
output.zip 是编译镜像的输出文件，其中包含有镜像，以及启动文件（正常不需要），可以烧录进TF卡中
main.py 是运行在个人主机上的文件。调用摄像头。建议使用PyCharm
